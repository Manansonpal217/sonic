# Assets Directory

This directory should contain the following assets for the app:

## Required Assets

1. **icon.png** (1024x1024px)
   - App icon for iOS and Android
   - Should be a square image with no transparency

2. **splash.png** (1284x2778px recommended)
   - Splash screen image
   - Will be displayed when the app launches

3. **adaptive-icon.png** (1024x1024px)
   - Android adaptive icon foreground
   - Should work on various Android launcher shapes

4. **favicon.png** (48x48px)
   - Web favicon (optional, for web builds)

## Generating Assets

You can use tools like:
- [Expo Asset Generator](https://www.npmjs.com/package/@expo/asset-generator)
- [App Icon Generator](https://www.appicon.co/)
- Design tools like Figma, Sketch, or Adobe XD

## Quick Setup

For development, you can use placeholder images temporarily. For production builds, replace with your actual branded assets.
